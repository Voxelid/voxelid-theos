Voxel XML Toolkit

A Full-featured toolkit for XML, XSLT and Schema Editing

Voxel XML Toolkit is an XML Editor written in Java as per the Rich Client Platform guidelines.

Voxel-X Rules Framework

Voxel-X is an editor for creating and running business rules

Voxel-X is an advanced editor for business rules that has in-built domain model generator, wizards, Plugins for running business rules

Voxel-R Rules For the AppEngine

Voxel-R is Voxel-X for the GAE Appengine

Written for an earlier version of GAE when it had no MySQL, Voxel-R uses the Appengine datastore to store the data

Voxel-P Layout

A framework for displaying and editing JavaBeans

Voxel-P Layout is a framework to display JavaBeans in a 1-cloumn or 2-column or custom layout, and subsequently edit the javabean using a form inside Java applications.

Voxel-H Cloud Connect

Voxel-H is a toolkit for making API calls to various cloud APIs

Voxel-H is a toolkit to call Amazon AWS cloud APIs and also Paypal APIs. Examples are available for AWS S3, DynamoDB APIs. Theres also some utilities for encrypting/decrypting. Works together with Voxel-Utils project for best results.

Voxel-Wizard

Voxel Wizard is a framework for creating wizard panels

Written for the Echo2 framework but can easily be ported to other frameworks, the Voxel Wizard is a full fledged framework for generating wizards in your java applications. Unlike ordinary wizard toolkits provided by other vendors, the Voxel Wizards are very robust and easy to use. There's other good stuff like being able to run asynchronous tasks via eclipse core runtime. You can use it along with some server push API like Comet, Atmosphere etc. for best results.

Voxel-D RIA Rules Engine

A web based platform for defining and executing business rules

Voxel-D is a web based platform written using the Echo2 framework for defining and running business rules. Voxel-D is a full fledged platform with support for built-in Domain modeler, Advanced rules authoring environment, multiple runtime views e.g Dynamic view and wizard view etc. Also includes a Beans Layout editor, a bindings editor, a workflow editor, a global settings editor, projects editor, rules editor, factstore editor and more. Voxel-D is by far the most advanced business rules management system ever developed. Use contact details below to license the Voxel-D BRMS software today.

Voxel-Bind Binding table

A support library for Voxel-D RIA to add EditableTable functionality

Often times there is a need to edit table data as it is displayed without loading another screen. This can be quiet easily implemented in a Java desktop environment. Web applications until recently did not have this functionality. The Voxel-Bind class library adds Editable Tables, combo boxes, textfields etc. to the Voxel suite.

Voxel-Misc Toolkit

A set of utilities for the Voxel-Rules

A set of classes for accomplishing tasks such as inspecting schemas, adding autocomplete to textfields and a names generator.

Voxel-S Ajax Shopping Cart

An Ajax Shopping cart component for Joomla Web CMS

A feature rich Ajax shopping cart component for the Joomla CMS written in PHP. Makes use of jQuery UI Mobile and the iLightBox script to display the ajax cart in a lightbox popup that is mobile friendly. Component has both front-end and admin modules and is best suited for small/medium sized businesses who offer both physical products as well as subscription based services. Multiple payment gateways supported via plugins and plugins provided for PayPal, Authorize.NET, PayUMoney, CCAvenue etc.

Voxel-Megha Multi-Cloud One Button Connect

A Multi-cloud One Button Connect component for Joomla Web CMS

Cloud computing is a hot topic these days. Many users have accounts with multiple cloud providers and typically runs workloads on servers from multiple vendors. The Multi-cloud One Button Connect application supports creating and running servers at over 40 different cloud providers e.g Aws, Rackspace, Digital Ocean etc. This component works along with the Voxel-Chef Module to provision pre-configured server instances at the click of a button.

Voxel-Util Utilities for the cloud

A set of PHP scripts for various operations on the cloud

Business applications run on the cloud. Business websites run on ISP provided servers on websites that run using PHP ad Mysql. The Utils package provides a set of PHP scripts for retrieving data from Mysql tables and uploading them to cloud datastores e.g AWS S3, CouchDB etc. using JSON format. Business Applications can use the Voxel-Cloud-Connect library to retrieve the data from cloud datastores and use the data.

Voxel-DevOps Chef for websites

A Chef DevOps package for deploying websites in an instant

DevOps is the art of setting up servers and applications in the datacenter or the cloud. Chef is a DevOps package written in Ruby for managing deployments as a unit. The Voxel-DevOps for websites package is designed to setup PHP/Mysql websites in an instant.

Voxel-Devops Chef for Applications

A Chef DevOps package for deploying websites in an instant

Voxel-DevOps for applications is a versatile package for deploying complete application stacks along with all dependent tools/services etc. on a single node on to a cloud server. The package managers all tasks such as installing tomcat web applications, copying files to anywhere on the filesystem, downloading and installing tarball/zipped packages, installing database servers and running sql scripts etc. Only requires Ruby to be installed which it installs itself. Runs well on Ubuntu servers.

Voxel-Lit Hundred Year's of technological Advancements - Part I

Coming Soon - A Novel in Non-fiction format

A first hand account of the hundred years of technological innovations from 1916-2016 from the perspective of a young professional who was there to see most of the innovations as they took place such as the birth of the world wide web, the java programming language, multi core processors, the advent of cloud computing etc.

Voxel-Lit Hundred Year's of technological Advancements - Part II

Advancements in XML and related technologies

XML may be derided these days after the arrival of JSON and annotations. XML is still very much prevalent and will play a significant role in the advancement of mankind. This book captures the gory details of how XML and its related technologies like XSLT, XML schema etc. This book is meant for the practitioners of XML and is not an easy read. If you wish to know more about XML and its growth, you can definitely buy this book. A webservices primer is included as a bonus.

Voxel-Glyph 3D Visualization Toolkit

A Visualization front-end for decision management systems

'Menaka' is a toolkit for rendering graphics on to a media like monitor screen etc. Voxel-Glyph is a 3D graphics visualization program that renders a Rubik's cube in 3D. The Cube also supports solving by allowing the cube sides to be rotated individually by the user. The cube can also be linked to a backend cube solving program (not included) to allow the programmatic solving of the Rubik's cube. This program is written in Menaka Java although Menaka-Ruby and Menaka-Javascript modes are also available.

### Advanced XML Framework

Our platform is a good way to learn XML. You will know what SAX, DOM, XSLT etc are and how to go about going from SAX to DOM or vice versa. You will know how to use JAXP to load differnt implementations of SAX/DOM parsers and XSLT transformers. You can learn about Source and Result interfaces and the various types of source/result objects e.g SAXSource, StreamSource, DOMResult etc. Plus many more

### Rich Client Platform

The Voxel appliction is a prototype for a rich client platform with support for creating menus, editors, wizards, syntax highlighting, drag and drop etc. Its very easy to learn and does not require a steep learning curve to get started

### AppEngine PaaS

This project works with the AppEngine PaaS platfrom from Google and makes extensive use of the Appengine SDK and the datastore to load/store data from AppEngine's datastore, which is a High Replication, BigTable NoSQL datastore.
